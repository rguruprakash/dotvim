# Installation

```
git clone http://github.com/rguruprakash/dotvim.git ~/.vim
ln -s ~/.vim/init.lua ~/.config/nvim/init.lua
ln -s ~/.vim/init.vim ~/.config/nvim/init.vim
ln -s ~/.vim/cheatsheet.txt ~/.config/nvim/cheatsheet.txt
ln -s ~/.vim/lua ~/.config/nvim/lua
```

```
## Install External dependencies (OSX) ##
brew install fzf
brew install the_silver_searcher
```
